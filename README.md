# ADK MCP Lab

Repository to explore and learn the Model Context Protocol (MCP) using the Google Agent Development Kit (ADK).

## Goals
- Understand MCP concepts
- Build simple MCP servers
- Integrate MCP with ADK agents
- Provide small reproducible examples

## Structure
- `examples/` – small MCP experiments
- `servers/` – MCP server implementations
- `agents/` – ADK agents using MCP
